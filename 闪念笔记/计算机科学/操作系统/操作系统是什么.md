## 闪念
操作系统是物理层上面的一层抽象，其提供了标准、统一、简单的 API 用来对物理层的硬件做统一的操作，方便了上层软件的开发，广义来说：微信小程序也是一种操作系统，它以系统作为基础，在其上层提供了更简单的 API 用来调用系统的硬件。
